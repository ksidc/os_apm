MyISAM 튜닝 전환 스크립트

DB 튜닝 스크립트 실행 후 사용하세요.

실행후 DB 재실행 해주세요

DB 튜닝 스크립트는 InnoDB 중심 튜닝으로 되어 있어 MyISAM 중심 튜닝으로 변경이 필요할때 사용 하시면됩니다.

권장 버전: MySQL 5.5, MariaDB 10.1 ~ 10.2


주의: MySQL 5.6, MariaDB 10.3 이상 버전은 InnoDB 기반으로 만들어져있어 기본 DB 튜닝 스크립트 결과로 나온 설정을 변환 없이 그대로 사용하는게 MyIsam 테이블에서도 성능이 더 잘 나옵니다.
